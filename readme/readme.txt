В папке requirements находится список необходимых библиотек и их версии
Установить их можно следующей командой:
pip install -r base.txt

В случае, если выдает ошибку связанную с библиотекой werkzeug следует удалить ее и установить более раннюю версию используя следующие команды:
pip uninstall werkzeug
pip install werkzeug==0.15.6

Запуск программы осуществляется командой:
python run.py
